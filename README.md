# 🛒 ASR Shopping Cart App

Welcome to the **ASR Shopping Cart**, a modern e-commerce application built using **Angular 19** and styled with **Bootstrap 5**. This app is a personal project designed to practice front-end development, Firebase integration, and real-time user interaction.

## 🌐 Live Demo

🔗 [Visit Live App](https://asshopping-app.web.app/)  
📦 [View on GitHub](https://github.com/ZmartAnand/Shopping_Cart)

---

## ✨ Features

- 🏠 **Home Page** with dynamic product listing  
- 🛍️ **Cart System** for adding/removing items  
- ❤️ **Wishlist Page** to save favorite products  
- 🙋‍♂️ **User Authentication** (Sign Up / Login / Reset Password)
- 🔐 **Edit Profile** and update user details  
- 📦 **Responsive Design** for mobile & desktop  
- 🔁 **Realtime Database** using Firebase  
- 💬 **Toasts** and smooth UX/UI elements

---

## 🛠️ Tech Stack

- **Angular 19** (Standalone Components)
- **Firebase Authentication** & **Firestore Database**
- **Bootstrap 5** for UI styling
- **ngx-intl-tel-input** for phone input with validation
